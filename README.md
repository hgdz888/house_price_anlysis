# house_price_anlysis
本项目依托合肥 106118 条二手房房源数据开展市场数据分析，基于 Python、Jupyter Notebook 工具，借助 Pandas 完成数据处理，Matplotlib、Seaborn 实现可视化分析。数据集共包含 12 个特征字段，涵盖房源所在片区、建筑面积、总价、单价、户型、朝向、建成年份、小区名称等关键信息。项目先导入数据分析依赖库并配置中文显示，读取本地房源数据集后，初步完成数据体量统计与样本预览，整体收录合肥多个热门小区二手房源，房屋面积跨度广，户型以刚需两室、三室为主，建成时间集中在 2013 至 2019 年。后续可通过数据清洗、特征提取、多维度统计可视化，探究户型、朝向、楼龄等因素对房价的影响客观剖析合肥二手房市场价格分布特征，为购房决策房产行业市场研究提供数据支撑
