一、项目简介
本项目基于合肥二手房公开销售数据集，使用 Python（Pandas、Matplotlib、Seaborn）完成数据导入、探索性分析、可视化统计，从房源基础概况、房价分布、户型、地段、建筑年份、单价等多维度解析合肥二手房市场现状，挖掘房产价格影响因素，为购房者、房产从业者提供数据参考。

2. 字段含义说明
表格
字段名	字段说明
city		城市（本次数据均为合肥市）
address	房源详细地段、道路位置
area		房屋建筑面积（单位：㎡）
floor	所在楼层及总楼层信息
name	小区名称
price	房屋总售价（单位：万元）
province	所属省份（安徽省）
rooms	户型（如 3 室 2 厅、2 室 1 厅）
toward	房屋朝向（南向、南北向等）
unit		房屋每平米单价（元 /㎡）
year		楼栋建造竣工年份
origin_url	房源原始爬取网页链接

3. 样例数据预览
数据集主要收录合肥各大片区二手房，包含圣地亚哥、正荣・悦都荟、文一名门金隅等热门小区，房源建筑面积跨度从 37㎡小户型到百平以上改善型住房，总价区间跨度大，建成年代覆盖 2013–2019 年，朝向以南北向、南向主流刚需朝向为主。

四、数据分析流程
步骤 1：环境初始化与第三方库导入
导入数值计算、数据处理、可视化相关库，配置中文字体解决图表中文乱码问题，保证后续可视化图表可以正常展示中文标题、标签。
步骤 2：数据集读取
通过 Pandas 的read_csv()函数读取本地 CSV 格式二手房数据集，完成数据加载，生成 DataFrame 结构化数据表用于后续分析。
步骤 3：数据基础概况探查
统计数据集总房源条数、总特征字段数量，快速掌握数据体量；
使用df.head(5)查看前 5 行样本，校验字段格式、数据内容、缺失异常情况；
初步观察房源所在区域、面积、总价、单价、户型、建造年份等核心业务字段的数据格式，为后续数据清洗、特征加工做准备。
五、后续可拓展分析方向
1. 数据清洗环节
缺失值、重复值检测与删除（样例可见存在重复房源数据，需去重）；
字符串字段数值提取：从area提取纯面积数字、price提取总价数值、unit提取单价、year提取建造年份；
楼层、户型、朝向分类特征规范化处理。
2. 描述性统计分析
房屋总价、单价、建筑面积的均值、中位数、最大最小值、四分位数分布；
不同户型、朝向、建成年份下的房价差异统计；
各热门小区、片区房源数量与均价排名。
3. 可视化分析方向
房价、房屋面积分布直方图 + 箱线图，识别高价异常房源；
不同户型、朝向的均价对比柱状图；
建筑建成年份与房价、单价相关性散点图；
各片区房源数量热力图、小区房源数量排名条形图；
面积 - 总价线性相关性拟合分析。
4. 深度挖掘方向
刚需小户型、改善型大户型房价特征差异分析；
楼龄（建成年份）对二手房单价的影响规律；
南北通透、南向等优质朝向对房产溢价的影响程度；
热门商圈、近郊片区房产价格梯度分析。
六、项目价值
对购房者：直观了解合肥二手房整体价格区间、热门片区、主流户型与合理单价区间，规避高价接盘风险；
对房产中介：定位热门小区、刚需主力户型，精准把握市场供需结构；
数据分析学习场景：完整复现从数据加载、探索、清洗、可视化到特征挖掘的数据分析全流程，适合 Python 数据分析实战练习。
